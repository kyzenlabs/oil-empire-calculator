# Oil Empire Calculator

A modern, fast, and fully client-side calculator for optimizing your progress in Oil Empire.

Originally inspired by the desktop application by h.263 (nitro.), this version provides a lightweight web-based experience with real-time calculations and no installation required.

---

## 🚀 Web Version

👉 https://kyzenlabs.github.io/oil-empire-calculator/

---

## ⚡ Features

### 📊 Production Stats
- Petrol per hour calculation
- Cash per hour calculation
- Real-time production updates
- Adjustable:
  - Cash per unit ($1–$15)
  - Boost multiplier (100%–285%)

---

### 🎯 Goals & Planning
- Time to reach a specific cash goal
- Drill affordability calculator
- Refinery cost & planning
- Upgrade timing optimization

---

### 🛢 Profit Tools
- Gas profit calculator
- Supports:
  - `100k`, `2.5M`, `16.9B`
  - Commas and shorthand inputs
- Real-time total profit calculation

---

### 🌍 Customization
- 12+ supported languages
- Accent color customization
- Clean, modern UI

---

### 📐 Layout Designer (Coming Soon)
- Planned:
  - Grid-based base builder
  - Drill/refinery placement
  - Optimization visualization

---

## 🧠 What This Solves

Oil Empire becomes heavily time-dependent in later stages.

This tool removes the need to:
- Manually calculate upgrades
- Guess profitability
- Use external calculators repeatedly

---

## 📦 Tech Stack

- HTML (single-page app)
- CSS (custom design system)
- Vanilla JavaScript

## 🌐 Hosting

This project is deployed using GitHub Pages.

To host your own version:

1. Fork the repository
2. Go to **Settings → Pages**
3. Select:
   - Branch: `main`
   - Folder: `/ (root)`
4. Save
